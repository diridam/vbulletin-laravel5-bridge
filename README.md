# Vbulletin 4.2.3 Bridge for Laravel 5.1
VBulletin 4.2.3 Bridge for Laravel 5.1

NOTE: this is a bridge to verify if the user is logged in or not.

Installation: 
1) add "diridam/laravel5vbbridge": "dev-master" to your composer.json
```
"require": {
		...
		"diridam/laravel5vbbridge": "dev-master"
	},
```
2) composer update
